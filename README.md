# luqueanguita_maria_ads

TIC TAC TOE IN C

TO RUN IT:

in Developer Command Prompt for VS 2017:

```
$ cd to project folder

$ cl tictactoe.c

$ tictactoe
```

in Mac / Linux:

```
$ gcc -o -lcurses tictactoe tictactoe.c

$ ./tictactoe
```
